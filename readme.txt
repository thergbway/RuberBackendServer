Parusa Backend server.

Данный сервер представляет собой приложение, которое необходимо развернуть на сервере приложений Wildfly.

Исходный код.
Исходный код сервера находится в открытом доступе и расположен на GitHub: https://github.com/thergbway/RuberBackendServer
В связи с активной разработкой сервера, версия на момент подачи на конкурс фиксирована: https://github.com/thergbway/RuberBackendServer/tree/dc06f469118fab2ccbf0211d069faf0fefe895eb
Так же исходный код проекта приложен к заявке на конкурс.

Сборка проекта.
Проект должен быть собран при помощи Maven 3.2.1 командой mvn package из корневой директории пакета. В результате в папке target будет находится /war файл.

Размещение на сервере.
Полученный .war архив веб-приложения необходимо разместить на сервере WildFly 9.0.2.Final и запустить его на выполнение.
Необходимо заметить, что на сервере должен быть правильно сконфигурирован источник данных. Необходимо использовать PostgreSQL 9.1.20.
Источник данных запрашивается приложением под jndi-именем "java:/RuberPostgresDS" с expected-type="javax.sql.DataSource".
Пожалуйста, правильно сконфигурируйте данный источник данных.

REST API.
REST API сервера, который он предоставляет клиентам описан на Github: https://github.com/thergbway/RuberBackendServer/wiki/API

Работаюший экземпляр сервера доступен по адресу: http://185.12.95.60:8081/ruber/

Используемые технологии: Java 8, Maven, Git, Spring MVC, Hibernate ORM, Jackson, PostgreSQL, JTA.
